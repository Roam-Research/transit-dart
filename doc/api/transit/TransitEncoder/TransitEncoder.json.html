Returns a TransitEncoder for emitting transit-formatted JSON objects.

This converter is meant to be chained with a JsonEncoder in a pipeline that converts a Dart native object into a JSON string, as illustrated below.

[Dart native object] ==(a)==> [formatted value] ==(b)==> [JSON string]

Above, the initial conversion (a) from a Dart object into a transit formatted value is handled by a TransitEncoder and the subsequent conversion (b) to a JSON string is handled by a JsonEncoder.

Here is an example of using a TransitEncoder and JsonRepeatEncoder (which processes a sequence of JSON objects) to encode individual objects to stdout.

 var objects = ['A', {null: null, 'foo': true}, 3.14];
 Stream
     .fromIterable(objects)
     .transform(TransitEncoder.json())
     .transform(JsonCombiner())
     .transform(utf8.encoder)
     .pipe(stdout);

Supply custom handlers as a map in customHandlers.
